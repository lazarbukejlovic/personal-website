# Lazar's Personal Website

Premium personal engineering website built to present my work, experience, resume, and contact flow in a more serious and product-oriented way than a typical portfolio template.

## [Live Demo](https://personal-website-tau-sandy.vercel.app/)

## Overview

Lazar Digital Foundation is a personal/professional website designed around a simple goal: present experience, selected work, and product direction in a clean, premium, and structured way.

Instead of feeling like a generic portfolio page, the project is built more like a real product surface:
- a clear professional identity
- experience-first presentation
- integrated resume view and PDF access
- direct project linking
- booking flow for intro calls
- polished visual system with consistent spacing and hierarchy

## What this project includes

- Premium hero section with direct professional positioning
- Experience-first structure instead of a generic “projects only” layout
- Dedicated resume page with downloadable PDF
- Booking / intro call flow with backend persistence
- Booking confirmation flow prepared through backend/email infrastructure
- Selected Work section linking directly to live demos and GitHub repositories
- Working external links for GitHub, LinkedIn, and email
- Responsive layout built for desktop and mobile use
- Clean visual style focused on clarity, structure, and credibility

## Why this project matters

This project is important because it presents not only finished work, but also how I position myself professionally.

It moves beyond a standard portfolio and acts more like a personal product layer:
- it connects experience with shipped work
- it shows full-stack thinking
- it presents contact and scheduling in a real application flow
- it keeps the design serious, simple, and product-oriented
